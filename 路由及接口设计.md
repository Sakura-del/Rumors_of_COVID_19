#### 列出谣言

---

###### 请求消息

```python
GET 	/rumor/views	 HTTP/1.1
```

###### 请求参数

http 请求消息url 中需携带以下参数，

* action

  必填项，填写值为list_rumors

###### 响应消息

```python
HTTP/1.1 200 OK
Content-Type:application/json
```

```python

              'markstyle': 'fake',
```

5. sourceurl：消息来源网址

